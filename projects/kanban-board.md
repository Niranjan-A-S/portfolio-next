# Kanban-Board

A Kanban board that simplifies the organization and tracking of tasks with filter options and drag-and-drop functionalities.

## Tech Stack

The project is built using the following technologies:

- TypeScript
- React
- React-Router-DOM
- Redux Toolkit
- Styled Components
